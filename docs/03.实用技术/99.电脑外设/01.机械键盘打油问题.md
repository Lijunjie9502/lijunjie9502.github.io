# 机械键盘打油问题

* 一般是由于键帽是 ABS 材质的原因， PBT 键帽不会出现这种问题
  * 所以可以换键盘帽来解决
* 但是查了一下 FILCO 键盘键帽似乎是经过人工配对来保证手感一致的，虽然有点玄学，但相比于手感，打油这种外观上的东西就显得没有那么重要的
* 最终解决办法还是多洗手，多擦键盘吧
