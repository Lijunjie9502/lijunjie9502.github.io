# vuepress 自定义css样式

:::note 方法
* 在 `docs/.vuepress/styles/index.styl` 中添加相应的 css 样式
:::tip
* `docs/.vuepress/styles/` 下的格式文件优化级较高
* 不要直接在主题中修改，不然后续更新时，容易被覆盖
:::

::: warning
注意 `.styl` 中样式的定义语法
:::

::: tip 配色网站可参考
* [flat ui colors](https://flatuicolors.com/palette/cn)
:::
