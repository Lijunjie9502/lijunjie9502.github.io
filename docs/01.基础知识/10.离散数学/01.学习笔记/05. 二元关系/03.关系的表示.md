## 关系的集合表示

::: tip
关系是一种特殊的集合，因此集合的两种基本表示法 (**枚举法和叙述法**)，可以用到关系的表示中
:::

::: details 示例
* 集合 $A = \{1, 2, 3, 4\}$ 上的整除关系 $R$ 可用枚举法表示为
  * $R = \{< 1, 1 >, < 1, 2 >, < 1, 3 >, < 1, 4 >, < 2, 2 >, < 2, 4 >, < 3, 3 >, < 4, 4 >\}$
* 实数集 $R$ 上的“相等”关系 $S$ 可用叙述法表示为
  * $S = \{< x, y > |(x, y ∈ R) ∧ (x = y)\}$
:::
