谓词符号化举例

# 谓词符号化举例

# 示例一: 使用一元谓词

没有人登上过木星；

  • 令 ： 是人，： 登上过木星,
    • 则命题符号化为
      • 或

在美国留学的学生未必都是亚洲人

  • 令 ： 是亚洲人，： 是在美国留学的学生
    • 则命题符号化为
      • 或

尽管有人很聪明，但未必一切人都聪明

  • 令 ： 是人；： 很聪明,
    • 则命题符号化为

# 示例二: 使用多元谓词

天下乌鸦一般黑

  • 令 ： 是乌鸦；： 与 一般黑
    • 则命题符号化为

每个实数都存在比它大的另外的实数

  • 令R(x)：x 是实数；L(x, y)：x 小于 y,
    • 则命题符号化为
    • 若假定个体域为所有实数，则命题符号化为

量词对变元的约束往往与量词的次序有关。不同的量词次序，可以产生不同的真值

  • 因此当多个量词同时出现时，不能随意颠倒它们的顺序，否则会改变原有的含义

# 示例三, 四: 限定个体域

限定个体域可以简化符号表示

符号化下列语句：

  • 所有狮子都是凶猛的
  • 有些狮子不喝咖啡
  • 有些凶猛的动物不喝咖啡

解

  • 令
    • : 是狮子
    • : 是凶猛的
    • : 喝咖啡
  • 假定所有动物的集合为个体域
    • 命题可符号化为

符号化下面一组语句：

  • 所有的蜂鸟都五彩斑斓
  • 没有大鸟以蜜为生
  • 不以蜜为生的鸟都色彩单调
  • 蜂鸟都是小鸟

解

  • 设
    • : 是蜂鸟
    • : 是大鸟
    • : 是以蜜为生的鸟
    • : 五彩斑斓
  • 假定所有鸟的集合为个体域
    • 则命题可符号化为
