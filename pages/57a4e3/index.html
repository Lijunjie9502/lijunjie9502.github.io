# LR 分析法的基本原理

自底向上分析的关键问题是正确的识别当前最右句型的句柄

  • 当前最右句型指的是栈内符号串 + 剩余输入
  • 句柄是逐步形成的，可以用“状态”表示句柄识别的进展程度
    • 每个项目集闭包对应一个状态
      • 项目集闭包 在后续章节引入
