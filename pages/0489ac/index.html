# 命题公式分类和等价

# 命题公式的分类

# 永真公式(重言式)

定义

  • 公式 称为永真公式(重言式,tautology)，如果在它的所有解释之下其真值都为“真”

# 永假公式(矛盾式)

定义

  • 公式 称为永假公式(矛盾式,contradiction)，如果在它的所有解释之下其真值都为“假”
    • 有时也称永假公式为不可满足公式

# 可满足公式

定义

  • 公式 称为可满足公式(satisfiable)，如果它不是永假的
